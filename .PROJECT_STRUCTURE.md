# Project Structure

```
back-plopsorteos/
│
├── config/                          # Configuración
│   ├── database.js                 # MongoDB connection
│   └── recaptcha.js                # reCAPTCHA v3 config
│
├── models/                         # Mongoose schemas
│   ├── PromotionalCode.js          # Schema de códigos
│   └── ValidationLog.js            # Schema de auditoría
│
├── controllers/                    # Lógica de negocio
│   └── validateCode.js             # Controller principal
│
├── routes/                         # Rutas del API
│   └── validate.js                 # Endpoints
│
├── middleware/                     # Middlewares Express
│   ├── logger.js                   # Logging
│   ├── errorHandler.js             # Error handling
│   └── validation.js               # Validación de entrada
│
├── utils/                          # Utilidades
│   └── recaptchaValidator.js       # Validación reCAPTCHA
│
├── scripts/                        # Scripts de utilidad
│   └── seedDatabase.js             # Poblar BD con ejemplos
│
├── logs/                           # Logs (generados por el app)
│   └── app.log                     # Log file
│
├── server.js                       # Entry point del servidor
├── package.json                    # Dependencias
├── .env                            # Variables de entorno (NO commitear)
├── .env.example                    # Template de .env
├── .gitignore                      # Git ignore
├── README.md                       # Documentación general
├── QUICK_START.md                  # Guía de inicio rápido
├── test-api.sh                     # Tests bash para Linux/Mac
├── test-api.ps1                    # Tests PowerShell para Windows
└── node_modules/                   # Dependencias (NO commitear)

```
